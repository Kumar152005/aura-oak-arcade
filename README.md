# Aura & Oak Arcade

A premium café-themed browser arcade with four complete games:

- Aura Snake
- Bean Catch
- Oak Memory
- Espresso Break

## Play

Open the GitHub Pages link in any modern browser. The arcade supports desktop keyboards, mouse input, and mobile touch gestures.

## Features

- Responsive mobile and desktop design
- Canvas-rendered game graphics
- Saved high scores using local storage
- Pause and restart controls
- No external libraries or assets

Built for Aura & Oak Coffee Café.
